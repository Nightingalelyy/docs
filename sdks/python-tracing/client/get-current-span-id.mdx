description: "Return the active span ID as hex string"
---

## Overview

Returns the current span ID or `None` when no span is active.

## Example

```python
from respan_tracing import get_client, task

@task(name="span_demo")
def span_demo():
    client = get_client()
    return client.get_current_span_id()

print(span_demo())
```
