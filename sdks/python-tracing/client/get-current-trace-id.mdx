description: "Return the active trace ID as hex string"
---

## Overview

Returns the current trace ID or `None` when no span is active.

## Example

```python
from respan_tracing import get_client, workflow

@workflow(name="trace_demo")
def trace_demo():
    client = get_client()
    return client.get_current_trace_id()

print(trace_demo())
```
