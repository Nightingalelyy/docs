---
title: "get_client()"
description: "Access the Respan client for span and trace operations"
---

## Overview

`get_client()` returns a `RespanClient` bound to the singleton tracer instance. Use it to read IDs, update the current span, add events, and control export.

## Usage

```python
from respan_tracing import RespanTelemetry, get_client

telemetry = RespanTelemetry()
client = get_client()

trace_id = client.get_current_trace_id()
span_id = client.get_current_span_id()
client.add_event("started")
```

## Returned type

- `RespanClient`
