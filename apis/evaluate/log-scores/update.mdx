---
title: Update log score
api: PATCH https://api.respan.ai/api/logs/{log_id}/scores/{score_id}/
---

Updates an existing score for a specific log.

<ParamField body="numerical_value" type="number">
  Optional updated numerical score value.
</ParamField>

<ParamField body="string_value" type="string">
  Optional updated string score value.
</ParamField>

<ParamField body="boolean_value" type="boolean">
  Optional updated boolean score value.
</ParamField>

<ParamField body="categorical_value" type="array">
  Optional updated categorical score value (array of strings). Use when evaluator's `score_value_type` is `categorical`.
</ParamField>

<RequestExample>
```Python Python
import requests

url = "https://api.respan.ai/api/logs/{log_id}/scores/{score_id}/"
api_key = "YOUR_KEY" # Replace with your actual Respan API key
data = {
    "numerical_value": 4.8,
    "string_value": "Excellent quality"
}
headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

response = requests.patch(url, headers=headers, json=data)
print(response.json())
```

