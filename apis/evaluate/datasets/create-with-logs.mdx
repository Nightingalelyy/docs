---
title: "Create dataset with specified logs"
api: "POST https://api.respan.ai/api/datasets/"
---

Create a dataset and specify which logs to include using filters.

## Authentication

- API key: `Authorization: Bearer <API key>`

## Request Body

<RequestExample>
```json Request
{
  "name": "My Dataset",
  "description": "Dataset description",
  "type": "sampling",
  "sampling": 50,
  "start_time": "2025-07-30T00:00:00Z",
  "end_time": "2025-08-01T00:00:00Z",
  "initial_log_filters": {
    "id": {
      "operator": "in",
      "value": ["log_id_1", "log_id_2"]
    }
  }
}
```
</RequestExample>

## Examples

<CodeGroup>

```python Python
import requests

url = "https://api.respan.ai/api/datasets/"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
    "name": "My Dataset",
    "description": "Dataset with specific logs",
    "type": "sampling",
    "sampling": 50,
    "start_time": "2025-07-30T00:00:00Z",
    "end_time": "2025-08-01T00:00:00Z",
    "initial_log_filters": {
        "id": {
            "operator": "in",
            "value": ["log_id_1", "log_id_2"]
        }
    }
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```

```bash cURL
curl -X POST "https://api.respan.ai/api/datasets/" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Dataset",
    "initial_log_filters": {
      "id": {
        "operator": "in",
        "value": ["log_id_1", "log_id_2"]
      }
    }
  }'
```

</CodeGroup>
