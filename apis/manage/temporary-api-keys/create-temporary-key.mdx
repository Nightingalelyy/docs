---
title: "Create API key"
api: "POST https://api.respan.ai/api/temporary-keys/"
---

You can create a temporary API key by sending a POST request to the temporary keys endpoint. Keys will not show in the platform.

## Body

<ParamField body="name" type="string" >
  The name of the temporary API key.
</ParamField>

<ParamField body="expiry_date" type="string" >
    The expiry date of the temporary API key. 
    
    It should be ISO format string, like `2025-03-15T01:19:33.263343Z`
</ParamField>

<ParamField body="max_usage" type="integer" default={-1}>
Number of times this key can be used for logging. -1 being default, means infinite times
</ParamField>

<ParamField body="rate_limit" type="integer" >
Calls per minute. This is going to be overridden by the plan's rate limit. 

Default is `Null`.
</ParamField>

<ParamField body="spending_limit" type="float" >
Spending with proxy. In US dollars

Default is `Null`.
</ParamField>

<ParamField body="is_test" type="boolean" >
Is this a test key or a prod key

Default is `false`.
</ParamField>
