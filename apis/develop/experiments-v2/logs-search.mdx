---
title: "Search experiment logs"
api: "POST https://api.respan.ai/api/v2/experiments/{experiment_id}/logs/list"
---

Search experiment logs using filters with the V2 API.

## Authentication

- API key: `Authorization: Bearer <API key>`

## Path Parameters

<ParamField path="experiment_id" type="string" required>
  The ID of the experiment to search logs for.
</ParamField>

## Request Body

<RequestExample>
```json Request
{
  "filters": {
    "comparison_key": {
      "operator": "in",
      "value": ["b7c84ae535cf66aef2f862eb260a14bd1aa9310220076cf196fec40860ac0d11"]
    }
  }
}
```
</RequestExample>

## Examples

<CodeGroup>

```python Python
import requests

experiment_id = "experiment_id_123"
url = f"https://api.respan.ai/api/v2/experiments/{experiment_id}/logs/list"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
    "filters": {
        "comparison_key": {
            "operator": "in",
            "value": ["b7c84ae535cf66aef2f862eb260a14bd1aa9310220076cf196fec40860ac0d11"]
        }
    }
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```

```bash cURL
curl -X POST "https://api.respan.ai/api/v2/experiments/{experiment_id}/logs/list" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "filters": {
      "comparison_key": {
        "operator": "in",
        "value": ["key_123"]
      }
    }
  }'
```

</CodeGroup>
