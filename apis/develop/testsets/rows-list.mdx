---
title: "List testset rows"
api: "GET https://api.respan.ai/api/testsets/{testset_id}/rows"
---

List all rows in a testset.

## Authentication

- API key: `Authorization: Bearer <API key>`

## Path Parameters

<ParamField path="testset_id" type="string" required>
  The ID of the testset to list rows from.
</ParamField>

## Examples

<CodeGroup>

```python Python
import requests

testset_id = "testset_id_123"
url = f"https://api.respan.ai/api/testsets/{testset_id}/rows"
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}

response = requests.get(url, headers=headers)
print(response.json())
```

```bash cURL
curl -X GET "https://api.respan.ai/api/testsets/{testset_id}/rows" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</CodeGroup>
