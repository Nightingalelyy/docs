Respan is committed to GDPR compliance and protecting privacy rights of individuals in the European Union.

## Data subject rights

We support all GDPR rights:

- **Access**: Request access to your personal data
- **Rectification**: Correct inaccurate data
- **Erasure**: "Right to be forgotten"
- **Restrict Processing**: Limit data processing
- **Data Portability**: Export data in machine-readable format
- **Object**: Opt-out of processing

## Data protection

- **Encryption**: Data encrypted at rest and in transit
- **Access Controls**: Role-based authentication
- **Data Minimization**: Only collect necessary data
- **Retention Policies**: Automated data lifecycle management
- **EU Data Residency**: Available upon request

## Legal basis

- **Contract Performance**: Service delivery
- **Legitimate Interests**: Platform security and improvement
- **Consent**: Optional features and communications

## Data processing agreement

Customers can execute a Data Processing Agreement (DPA) covering processing instructions, security measures, and breach procedures.
